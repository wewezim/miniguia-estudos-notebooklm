# miniguia-estudos-notebooklm
Caderno Temático no NotebookLM com algum tema de estudo como forma de cria uma atividade prática

Contexto: esse material de estudo serve como uma tarefa prática dos conhecimentos adquiridos na aula da DIO sobre Treinando uma IA de Aprendizagem: Explore o Poder do NotebookLM, em que será criado um mini guia de estudos em uma área específica

Objetivos: cria um mini guia de estudo sobre como criar uma dieta saudável para uma adulto de 40 anos, o qual está com um sobrepeso e precisa diminuir a balança para entrar em forma. Este guia conterá também um apoio de como gerenciar as compras dos materiais para realizar a dieta.
